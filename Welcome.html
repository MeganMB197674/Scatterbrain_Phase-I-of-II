Overview & Instructions:

Welcome to 'Scatterbrain!' Scatterbrain is a memory game which aims to test how well you can remember a given set of data. In this game, you will be provided a sentence or phrase for a specified amount of time. Then, once the timer is up, some words will magically disappear from the sentence or phrase and you will be asked to select the correct words from a drop-down menu.
